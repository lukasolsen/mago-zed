# Agents.md

Mago Zed is a Zed Extension (a IDE written in Rust) that provides a set of tools to help PHP code styles using Mago.

This is a Rust Workspace that contains the following crates:
- `cli`: A command line interface to run Mago and parse its output.
- `lsp-server`: A Language Server Protocol (LSP) server that provides code analysis and formatting features to Zed.
